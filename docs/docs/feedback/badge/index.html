PropertyTypeDescription
classstringAdditional custom class names to apply to the badge component.
styleCSSPropertiesInline styles to apply to the badge container. Defaults to { height: 0, width: 0 }.
sizenumberThe size of the badge's font. Default is 15.
sizeType'rem' | 'rm' | 'px' | stringSpecifies the unit for the badge size. Default is 'px'.
position'top-right' | 'top-left' | 'bottom-right' | 'bottom-left'Position of the badge relative to its container. Default is 'top-right'.
shape'circle' | 'square' | numberThe shape of the badge. Can be 'circle', 'square', or a numeric value representing a border radius percentage. Default is 'circle'.
variant'solid' | 'outlined' | 'text'The visual style of the badge. Default is 'solid'.
color'success' | 'error' | 'warning' | 'accent' | 'primary'The color theme of the badge. Default is 'primary'.
